🚀 COMO EXECUTAR

1. SETUP INICIAL
   pip install -r requirements.txt
   cp .env.example .env  # Configure Supabase + DagsHub

2. OPÇÃO A: Automático com DVC (Recomendado)
   dvc repro

3. OPÇÃO B: Manual (passo-a-passo)
   python src/ingestion.py         # Fetch dados
   python src/preprocessing.py     # Feature engineering
   python src/prepare_data.py      # Split + preprocessor
   python src/train.py             # Treina 12 modelos
   python src/evaluate.py          # Avalia no test set

4. UI INTERATIVA
   streamlit run streamlit_ui.py
   → Abra http://localhost:8501

5. API REST (Opcional)
   python main.py
   → Endpoint POST http://localhost:8000/predict
